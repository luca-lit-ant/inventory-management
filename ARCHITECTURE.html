Request Data Flow

User changes filter
FilterBar.vue binds to module-scoped refs in useFilters.js — state shared across all views.
View watcher fires
Each view watch()es filters and calls its loadData() function on change.
API client builds request
api.js translates filter object → URLSearchParams, skipping 'all' values, then calls axios.
FastAPI filters in-memory
apply_filters() + filter_by_month() run Python list comprehensions over module-level lists loaded from JSON at import time.
Pydantic validates response
response_model=List[Model] enforces schema on the way out — structural drift between JSON and models fails here.
View stores + derives
Raw response lands in a ref; display values (totals, groupings, chart data) are computed from it.
